To open the program:
    1. download and install any version of python 3 between 3.12-3.13.
    2. install the [illium.ttf] font.
    3. install guizero to your home dir.
    4. open [loger_file_editor.py] on the desktop or in a terminal.
        a.if a terminal window opens  too, check to see if the debug flag is enabled in [runtimeOptions.properties]
    5. your saves are in [projects\]

To modify in any way:
    1. go to the github repo and fork the project.
        a. get coding and make a great mod, who knows it may become part of the project.
    2. include atribution in your credits!

The code is in [gui.py]
All depenceies other than python and guizero are included and located in [libraries\].
All themes are in [assets\] along with other images.
